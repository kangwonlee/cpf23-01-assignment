## Github 온라인 편집기 사용법

* <kbd>.</kbd> 키를 누르면 MS VS Code 의 Web version 이 시작됨
* 수정 사항을 commit 등록 하려면 왼쪽에서 줄 셋 아래 (확대경 다음) 세번째 가지치기 아이콘 선택
* 파일 이름의 오른쪽 + 기호 선택 (staging)
* 위 빈칸에 변경 사항 설명 입력
* [커밋 및 푸시] 선택
* 줄 셋 의 [리포지토리로 이동] 선택하여 저장소로 복귀
